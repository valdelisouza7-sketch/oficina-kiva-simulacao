# Identidade visual (theme)

Esta pasta define cores, fontes e estilos da loja.

A identidade visual não depende dos produtos.
Ela garante consistência em toda a vitrine.

Nesta etapa:
- observe que o visual tem um lugar próprio
- perceba por que mudanças aqui afetam todo o sistema

Não edite nada.
Apenas entenda a lógica.
