# Proteção do sistema (snapshots)

Snapshots registram o estado da loja antes de publicações.

Eles existem para permitir voltar atrás com segurança
caso algo dê errado.

Nesta atividade:
- observe que o sistema se protege antes de mudar
- perceba que errar é previsto, mas precisa de controle

Segurança vem antes da publicação.
