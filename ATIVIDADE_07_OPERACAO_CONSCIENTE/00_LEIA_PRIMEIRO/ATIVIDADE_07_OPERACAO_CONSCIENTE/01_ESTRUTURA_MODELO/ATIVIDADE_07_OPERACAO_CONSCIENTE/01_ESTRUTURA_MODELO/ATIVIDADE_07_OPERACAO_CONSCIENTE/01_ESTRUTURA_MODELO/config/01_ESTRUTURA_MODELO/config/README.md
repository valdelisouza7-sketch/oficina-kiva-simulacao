# Regras estruturais do sistema (config)

Esta pasta guarda regras que definem como o sistema funciona.

Aqui não é lugar de operação diária.
Alterações indevidas podem comprometer o funcionamento da loja.

Nesta atividade:
- observe que regras e dados ficam separados
- perceba que nem tudo deve ser acessível a quem opera o dia a dia

Você não precisa entender a estrutura técnica do arquivo JSON.
Ele está aqui apenas para ilustrar como regras são organizadas no sistema.
