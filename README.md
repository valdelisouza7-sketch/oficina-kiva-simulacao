# Oficina Kiva — Ambiente de Simulação

Este repositório contém um ambiente de simulação educacional utilizado na Oficina Kiva.

Ele foi criado para permitir que alunos tenham contato com a estrutura real de um projeto
de e-commerce inteligente, sem risco de edição ou publicação.

## Importante

- Este ambiente é SOMENTE LEITURA
- Nenhum arquivo aqui deve ser editado
- Nenhum dado é real
- Nada aqui será publicado

## Objetivo do ambiente

- Reconhecer a organização de um projeto real
- Entender o papel de cada pasta
- Aprender onde não se deve mexer
- Desenvolver postura profissional antes da prática real

## Como usar

Siga as orientações apresentadas no curso.
Este ambiente não exige conhecimento técnico ou programação.

Aqui, aprender começa pela observação consciente.
